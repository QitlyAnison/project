Язык программирования Python

На что способен Python :

    Работа с FTP
    Работа с http запросами
    GUI (графический интерфейс)
    Создание веб-сценариев
    Программирование математических и научных вычислений
    Робототехника
    Работа с изображениями, аудио и видео файлами
    Работа с xml/html файлами
Преимущества Python среди других языков :

    Высокая сокорость выполения программ. Это связанно с тем, что основные библиотеки Python написаны на C++.
    В связи с этим вы можете писать свои собственные модули для Python на C или C++.
    В стандартныx библиотеках присутствуют средства для работы с электронной почтой, протоколами Интернета, FTP, HTTP.
    Имеет поддержку кроссплатформенности.
    Подходит для любых решений в области программирования.

Как начать программировать на Python :

  1. Скачать компилятор для Python с оффициального сайта |оффицальный сайт|
  2. После установки запустить среду разработки ILDE
  3. Создать и сохранить новый файл
  4. Начать писать код

Создание первой программы :

  1. Запускаем IDLE (изначально запускается в интерактивном режиме), после чего уже можно начинать писать первую программу. Традиционно, первой программой у нас будет "hello world".Чтобы написать "hello world" на python, достаточно всего одной строки.
  2. Вводим нашу первую функцию : print("Hello world!")|print выводит аргумент заданный из скобок в программу|
  3. Радуемся и продолжаем дальше.


Более глубокое изучение Python на практике :

  1. Начнем как принято с основ :

    1. Для начала создадим программный файл :
      Чтобы создать новое окно, в интерактивном режиме IDLE выберите File → New File (или нажмите Ctrl + N).
    2. Создадим нашу первую переменную обозначив ее Name и запишем в нее имя пользователя в подобном виде :
      name = input("Введите ваше имя") |input означает ввод данных с клавиатуры , то есть от имя пользователя|
      |аргумент в круглых скобрах означает текст который выведется при вызове функции input|
    3. После того как мы создали переменную давайте выведем ее значение , все так же используем функцию print("аргумент") : print(Name) |в print можно использовать арифметические действия между аргументами|
    4. Проверим сложение аргументов в print , сложим 2 числа введенных пользователем :
      a = input("First: ") ; b = input("Second: ") ; print(a + b)
      |как не странно ответ у нас получился не утешительный |
      : Произошло это потому-что мы не указали тип нашей переменной.Что бы указать тип переменной нужно перед функцией input указать тип : number = int(input("First number"))
